A

由函数调用规约可知：x 和 n 分别存放在 rdi 和 rsi 中；result 显然存放在 rax 中。 然后就很明显了——mask 存放在 rdx 中。

B

movl $1, %edx

movl $0, %eax

result 初始值为 0，mask 初始值为 1。

C

testq %rdx, %rdx

jne .L3

如果 mask 不为 0，则继续循环。

D

movl %esi, %ecx

...

salq %cl, %rdx

每轮循环中，mask 都会左移 n 位。

E

movq %rdi, %r8

andq %rdx, %r8

orq %r8, %rax

每轮循环中，result 都会与 x&mask 进行或运算。
